# **Office365数据所属地区扫描器** by Kevin

## 使用

工具使用3个文本进行工作，分别为

```
account.txt    //用于存储需要扫描的O365账号
result.txt     //用于存储结果，可以使用Excel打开，以空格切割
error.txt      //出现错误的账号存储
```
这里需要注意的是account.txt必须存储的为xxx@xxx.onmicrosoft.com，否则会被丢弃入错误账号的区块。
